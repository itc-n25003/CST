# CTFとは
「Capture The Flug」の略で直訳すると旗取り合戦(ここで言う旗は情報を意味する)  
用意されたWebサイトなどからFlagと呼ばれる情報を探し出したりするサイバーセキュリティのスキルを競うコンテストのこと。  

### 競技形式について
Jeopardy形式とAttack-Defense形式の２つの形式に大きく分けられる。
* Jeopardyは出題される問題を解いていく形式(SEA/J CTF for Studentsはこちらの方式)  
* Attack-Defenseは与えられたサーバーの脆弱性を修正して他チームにフラグ(情報)を取られないように守りつつ、相手のサーバーの脆弱性を利用して相手の旗を盗み取る形式。

### 出題傾向について
様々な問題ジャンルがあり、大会により少し異なる場合がある。問題ごとに要する言語の知識が違う。 
* Reversing
	→ 与えられたバイナリを解析して隠されたFlagを見つけ出す。 
* Pwning
	→ 与えられたバイナリを解析してサーバの脆弱性を見つけ、Exploitコードを作成して侵入しFlagを盗み取る。  
* Web
	→ ウェブサイトを解析して脆弱性を見つけ出し、Flagを盗み出す。  
* Cryptography
	→ 暗号を解き、Flagを見つけ出す。パズルのようなものや古典的暗号、RSAやハッシュ関数攻撃を要するものもある。数論の知識や柔軟な思考力も問われる。

### 講義での要望及び目標
この講義では知識の獲得とともに応用方法も身に着け、可能な限り大会での上位を目指せるように励みたいです。  
そのために基礎知識に抜けがないか確認しつつ現在の知識量を把握し、未学習の言語の基礎や問題への向き合い方を学べることを望みます。  
また、体験談を見ると結果を出すためには問題に慣れる必要がありそうなので、時間のあいてる際に問題サイトで自習できればなお良いかなと感じました。
